# VChat

VChat，一款即时通讯 Web 应用。

## 文件结构

```bash
.
├── LICENSE
├── README.md
├── client                  # 客户端，React、Redux、React-Router、Styled-Component 构建
│   ├── README.md
│   ├── package.json
│   ├── public
│   ├── src
│   └── yarn.lock
└── server                  # 服务器端，Koa、MongoDB、Socket.io 构建
    ├── README.md
    ├── app.js
    ├── constants.js
    ├── helper
    ├── models              # MongoDB Collection 模型
    ├── package.json
    └── routers             # Koa-router
```
